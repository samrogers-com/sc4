DESCRIPTION

This is a factory-sealed hobby box of 1995 Krome Productions Betty Boop Series 1 Premier Edition trading cards — the inaugural set for Max Fleischer's iconic flapper-era cartoon star. The set was begun by Echo Productions and completed by Krome after Echo's financial difficulties, making it a notable piece of mid-90s non-sport hobby history. Each box contains 36 wax packs.
