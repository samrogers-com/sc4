# Star Wars Topps Trading Cards — Star System Guide

## The Star Rating System (1977-1983 Topps Sets)

The original Topps Star Wars, Empire Strikes Back, and Return of the Jedi trading card sets feature a unique **star rating system** on each card:

### How It Works
- Each card has either a **1-star (*)** or **2-star (**)** marking
- Cards with **1-star** have partial puzzle images on the back that, when all 1-star cards from a series are assembled together, form a **complete picture (Puzzle A)**
- Cards with **2-star** have a different set of partial puzzle images that form a **separate complete picture (Puzzle B)**
- **Mixed star sets** contain both 1-star and 2-star cards — these are incomplete puzzle sets but contain a mix of both card types

### Set Types
- **1-star set**: All cards are 1-star — completes Puzzle A on the backs
- **2-star set**: All cards are 2-star — completes Puzzle B on the backs
- **Mixed star set**: Contains both 1-star and 2-star cards — neither puzzle is complete

### Why This Matters for Selling
- **Complete 1-star or 2-star sets are MORE valuable** because the buyer gets a complete puzzle
- **Mixed star sets are less valuable** — buyer gets partial puzzles from both
- This is an important selling point for eBay listings — always specify the star type
- Collectors specifically seek out complete 1-star or 2-star sets

### Series Breakdown (Topps Star Wars 1977)
The original 1977 set was released in 5 series over the theatrical run:
- **Series 1** (Blue border) — Cards #1-66
- **Series 2** (Red border) — Cards #67-132
- **Series 3** (Yellow border) — Cards #133-198
- **Series 4** (Green border) — Cards #199-264
- **Series 5** (Orange border) — Cards #265-330

Each series also has a corresponding **sticker set** (numbered separately).

### Series Breakdown (Topps Empire Strikes Back 1980)
- **Series 1** (Red border) — Cards #1-132
- **Series 2** (Blue border) — Cards #133-264
- **Series 3** (Yellow border) — Cards #265-352

### Series Breakdown (Topps Return of the Jedi 1983)
- **Series 1** — Cards #1-132
- **Series 2** — Cards #133-220

## Notes for eBay Listings
- **ALWAYS specify star type** (1-star, 2-star, or mixed) in the title and description
- **1-star and 2-star complete sets** command a premium — highlight "COMPLETE PUZZLE ON BACK"
- **Mixed sets** should note "contains both 1-star and 2-star cards — partial puzzles"
- **Sticker sets** are sold separately from card sets
- **Series border color** helps buyers identify which series at a glance
- Include the series number and border color in the listing title

## Notes for Website
- Each series should show the star breakdown with quantities
- Allow filtering by star type (1-star, 2-star, mixed)
- Show which puzzle the backs form when possible
- Link to eBay listings for each available set
