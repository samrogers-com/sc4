DESCRIPTION

This is a complete 72-card base set from the 1997 Topps Star Wars Trilogy Special Edition Widevision series. Released to coincide with the theatrical re-release of the Special Edition trilogy, this set features 24 cards from each of the three original Star Wars films. Each film is distinguished by a unique color border — brown for A New Hope, blue for Empire Strikes Back, and green for Return of the Jedi. The oversized Widevision cards showcase scenes in their original theatrical aspect ratio.
